# Experiment 31: Character Frequency

## Problem Statement

Write a Java program to find the frequency of a given character in a string.

## Input Format

* First line: A string $S$.
* Second line: A character $C$.

## Output Format

* A single integer representing the count of character $C$ in string $S$.

### Example 1

**Input:**

```text
Hello World
l
```

**Output:**

```text
3
```

### Explanation
The character 'l' appears 3 times in "Hello World" (twice in Hello, once in World).
